Tourism Management System

A comprehensive web application for managing tourism services, bookings, and user experiences. Developed as a university project.

View on GitHub

Key Features

User Authentication

Secure login and registration system with user profiles and role-based access control.

Destination Management

Browse and search for travel destinations with detailed information and images.

Package Booking

Book travel packages with different options and manage your bookings.

Booking Management

View and manage your bookings with status tracking and cancellation options.

Admin Dashboard

Comprehensive admin panel for managing destinations, packages, users, and bookings.

Responsive Design

Works seamlessly on desktop and mobile devices with a modern, user-friendly interface.
